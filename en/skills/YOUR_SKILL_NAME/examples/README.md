# examples (optional)

Each example is a standalone Markdown file recording one real-tested input/output pair.

## Adding examples

Copy `example_01.md`, rename it (`example_02.md`, `example_03.md`…), and fill in the new scenario.

## Recommended count

| Skill complexity | Suggested examples |
|-----------------|-------------------|
| Single input/output | 1 |
| Multiple input scenarios | 2–3 |
| Has scripts or multi-turn dialogue | 3+ |
