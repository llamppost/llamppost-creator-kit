# Avatar（套装必备）

套装上架需要 Avatar。请在这个文件夹内放：

- `avatar.png` — 至少 512×512，建议 1024×1024，PNG 格式，1:1 比例
- `metadata.json` — 已预填 placeholder，请编辑

完整规格与版权政策见 [`docs/avatar-creation-spec.md`](../../../docs/avatar-creation-spec.md)。

## 没准备 Avatar 想先试水温？

把同层的 `persona.md` 跟 `SKILL.md` 搬到 `personas/` 跟 `skills/` 文件夹，当单品上架。等补好 Avatar 再重新以套装上架。
